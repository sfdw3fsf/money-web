# Binance Auto Trader - OpenClaw Skill

Dependencies cho hệ thống giao dịch tự động.

## Cài đặt

```bash
pip install -r requirements.txt
```

## Lưu ý
- Nếu `TA-Lib` gặp lỗi trên Windows, cài từ wheel:
  https://github.com/cgohlke/talib-build/releases
- Hoặc bỏ qua TA-Lib, hệ thống sử dụng pandas tự tính indicators
